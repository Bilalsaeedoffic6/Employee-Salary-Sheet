# 📊 Automated Payroll & Salary Sheet System

A fully automated, formula-driven Excel payroll tracking system designed for flawless salary calculation and corporate reporting.

## ✨ Key Features
* 👤 **Employee Profiling:** Track records via ID, Name, and Designation.
* 🕒 **Attendance & Overtime:** Automated prorated basic earnings and hourly OT calculation.
* 💰 **Allowances Covered:** Dynamic calculation of HRA, DA, and Conveyance.
* 📉 **Legal Deductions:** Auto-configured logic for PF and ESI compliance.
* 🎯 **Final Output:** Instant, error-free **In-Hand Salary** computation.

## 🛠️ Tech Stack
* **Tool:** Microsoft Excel / Google Sheets
* **Functions Used:** Dynamic Formulas, Basic Algebra Logic, Conditional Formatting

## 🚀 How To Use
1. Enter employee baseline data (ID, Name, Designation, Basic Salary).
2. Input monthly Attendance and Overtime Hours.
3. Allowances, Gross, Deductions, and final In-Hand Salary will compute automatically.
